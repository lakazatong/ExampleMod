# Example Mod

This template is for Minecraft 1.12.4+ and uses:
- forge 54.0.10
- neoforge 21.4.33-beta
- fabric 0.112.2
- parchment 2024.12.22

You can use `./rename_author.sh` and `./rename_mod.sh` to properly rename folders, files and files' content
